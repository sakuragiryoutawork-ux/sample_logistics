# Sample_Logistics(架空のコーポレートサイト)
## 概要

Web制作学習サイト「Web鍛」のコーディング課題を使用した模写コーディング作品です。
デザインカンプを元にHTML / CSSで実装し、PC・スマートフォンンのレスポンシブ対応を行いました。

制作時間

## 使用技術

・HTML
・CSS
・JavaScript

## 工夫した点

・画面幅に応じてナビゲーションの表示を切り替えるようにしました。
786px以下ではハンバーガーメニューを表示し、787px以上ではナビゲーションボタンが横並びで表示されるようにCSSのメディアクエリを使用して実装しています。

・物流センターカードのテキスト部分の高さが要素ごとにずれてしまう問題があったため、
display: flex; を使用して高さを揃え、カードの見た目が揃うように調整しました。
これにより、画面幅が変わってもレイアウトが崩れにくい構造にしています。

## 苦労した点

ハンバーガーメニューを実装する際、position: fixed; を指定したナビゲーションボタンが画面外に表示されてしまう問題がありました。
原因を調査したところ、別のセクションの画像が画面幅を超えて表示されており、その影響でレイアウト全体が横に広がってしまっていました。
画像サイズを調整することでレイアウトの崩れを修正し、正常に表示されるように改善しました。
